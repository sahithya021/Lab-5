Step 1: Open Project Folder and open the cmd in that location

Step 2: Create a virtualenv using the command (mkvirtualenv Project)

Step 3: Install django using the command (pip install django)

Step 4: Run the manage.py file using the command (python manage.py runserver)

Step 5: Copy the Http address(http://127.0.0.1:8000/) and paste it in the browser. then it will display the "Hello world" Message
